# 🕵️‍♂️ Basic Network Sniffer using Python + Scapy

This is a simple yet powerful network sniffer.

## 🔧 Features
- Live packet capturing using Scapy
- Displays:
  - Source IP
  - Destination IP
  - Protocol
  - Timestamp
- Supports filters: ICMP, TCP, HTTP, All
- Color-coded output using `colorama`
